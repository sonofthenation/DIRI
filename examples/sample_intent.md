# Developer Intent

I want to build a premium student planner web app.

It should not feel like a basic admin panel.
It should feel like an open notebook on a desk.
The interface should be calm, academic, polished, and visually rich.

Required features:
- dashboard
- tasks
- subjects
- notes
- quizzes
- schedule
- calendar
- hover previews
- iframe support

Must not feel like:
- plain Bootstrap
- boring grid
- empty cards
- cheap template
